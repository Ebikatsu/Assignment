横型探索(幅優先探索)
グラフ上の始点から終点までの経路を調べる方法。
節点を入れることができるキューと、接点から接点もしくはnullへの関数を用いる。はじめこの関数は任意の入力に対してnullを返す。

節点 経路[経路の長さ];
int i;

始点をキューに入れる。
while(キューが空でないかつキューの中に終点がない。)
{
	キューからひとつ取り出す。
	キューからとり出された節点の隣接点のうち、関数の写像にないものをキューに入れる。
	関数にキューに入れられた節点を入力した時の出力を取り出されたキューにする。
}
if(キューに終点が入っている)
{
	i = 0;
	経路[i] = 終点;
	while(経路[i] != 始点)経路[++i] = 逆関数(経路[i]);
}
