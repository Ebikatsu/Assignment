\section{課題}
働き方改革実現会議では先に(下記に)上げたような施策がリストアップされているが、各自は何が最も優先課題と考えるか。その理由を添えてレポートとしてまとめてください。
\begin{itemize}
	\item 同一労働同一賃金など非正規雇用の処遇改善
	\item 賃金引き上げと労働生産性の向上
	\item 時間外労働の上限規制の在り方など長時間労働の是正
	\item 雇用吸収力の高い産業への転職・再就職支援、人材育成、格差を固定化させない教育の問題
	\item テレワーク、副業・兼業などの柔軟な働き方
	\item 働き方に中立的な社会保障制度・税制など女性・若者が活躍しやすい環境整備
	\item 病気の治療、子育て・介護と仕事の両立
\end{itemize}
\end{comment}
\section{自分が考える最優先課題}
私は時間外労働の上限規制の在り方など長時間労働の是正が最優先課題であると考える。それによって自殺してしまうこともある、命に係わる問題であるからだ。今回はこの問題が生じる原因について調べ、解決するにはどうすればよいかを考えた。

\begin{comment}
\subsection{日本政府が働き方改革を進めている理由}
\begin{itemize}
	\item 日本の人口、特に労働力人口が継続して減少していること
	\item 日本の長時間労働がなかなか改善されないこと
	\item 政府が推奨しているダイバーシティーマネジメントや生産性向上が働き方改革と直接的につながっていること
\end{itemize}
ダイバーシティーマネジメントとは、個人の性別や人種、国籍などの違いにこだわらずに優秀な人材を活用する企業経営方式である。
安倍首相は働き方改革を、「（一億総活躍社会実現にむけた）（今後三年間の）最大のチャレンジ」と位置付けている。
企業の立場からは大きな負担になることもあるだろう。生産性向上や経済成長だけを優先すると、労働者の生活の質はより悪化する恐れが高い。労働者の健康や生活の満足度を優先的に考慮して実施されることを望むと主張。
\cite{nissei}
\subsection{二兎を追う}
労働問題は様々な立場の人たちの利害が直接ぶつかり合う。それだけに抜本的な改革にはなかなか着手できずに来た。企業の生産性を掲げて労働問題に取り組めば、労働側は強硬に反対する。労働条件改善に先に手を付けようとすれば、経営者団体から猛烈な反発を受ける。結局、双方が折り合う仕組みを目指してきたことで、社会保障費や助成金の増加という形で国家財政に大きな付けが回った。企業は稼ぐ力を失い、働き手は長時間労働と低賃金に苦しみ、国は経済成長力を失い借金まみれ。そこから抜け出すには、一気に「働き方」を抜本的に見直すことが必要だと安倍首相は強調しているのだ。
「働き方改革は、第三の矢、構造改革の柱となる改革」と安倍首相は述べた。
\cite{gendaibusiness}
\subsection{アベノミクス「旧三本の矢」}
\begin{itemize}
	\item 大胆な金融政策
	\item 機動的な財政政策
	\item 投資を喚起する成長戦略
\end{itemize}
大胆な金融政策は日銀の協力を得て円安・株高でアベノミクスの基盤を気付くことが出来たが、利害調整が難しい施策を後回しとなり、財政政策は一時的な刺激策でしかなく、市場が期待していた成長戦略は「道半ば」で終わった。

\end{itemize}

%\subsection{長時間労働対策について}
企業が長時間労働対策として、決まった時間になったらオフィスを消灯するなどの対策を実施したとしても、業務時間内に仕事を終えることが出来ずに、持ち帰り残業という新たな問題が発生し、根本的な問題解決にならないということも起きている。また、労働基準法上の規制が緩い管理監督者が、残業規制で仕事を終えることが出来ない部下の仕事を引き受けて長時間労働となることも起きている。
%\cite{toyokeizaizangyo}
単に労働の仕方に対して制限を設けたとしても、やるべき仕事の量が変わらなければ当然別の人、別の場所、別の時間でその仕事をしなければならず、根本的な問題解決にはならない。解決するべきなのは長時間労働そのものではなく、その根本にある仕事の量だろう。それは政府による政策や制度によってどうにかなるものではなく、日本人の働き方や文化を変える必要があるだろう。どの業種においても、顧客の立場が高く、企業は競争に勝つために顧客の要求に対して先回りして品質を高め、顧客は無意識にそれに甘えてしまっているという傾向が文化的な原因として考えられる。こうした傾向を持つ社会から脱するために、顧客側が意識を変える必要があるだろう。国民一人一人が、提供されるサービスに対して、そのサービスを実現するためにどれだけの人間がどれだけの苦労をしているのかをよく考え、感謝の心を持ち、無理な要求を控えることを意識することが解決への道となるだろう。
%\cite{toyokeizaiboso}
\section{考察}
今回働き方に関する問題、特に長時間労働についてや、その解決策としての働き方改革について調べ、長時間労働という問題は政府による政策や制度によって解決できるような簡単なものではないことが分かった。そして、日本の社会のこういった状況を解決するために、自分が顧客側の立場に立った時に感謝の心を持つことが大切であると気づくことが出来た。
