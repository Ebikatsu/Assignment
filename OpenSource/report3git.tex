\subsection{ARとは}
ARとは、現実世界で人が感知する情報に、「何か別の情報」を加えて表現する技術である。具体例として、
\begin{itemize}
	\item 夜空にかざすと、見えている星座を確認することができるアプリ\\
		ARアプリの王道ともいえるオーソドックスな取り組み
%		\cite{DifferenceBetweenVRAndAR}
	\item ショッピングセンターの大型スクリーンを利用した、ナショナルジオグラフィック社のARプロモーション$<$https://vimeo.com/40442188$>$\\
		ショッピングセンターのような公共性の高い場所で、大型スクリーンを活用した多人数参加型・体験型のARコンテンツで、都市空間での現実世界に、大自然やロストワールドの世界映像を重畳させるというダイナミックな取り組みで、世界中で開催され話題を集めた。\\
		ARは企業のプロモーションやキャンペーンとも親和性が高く、様々な事例が存在する。
%		\cite{DifferenceBetweenVRAndAR}
	\item ARによって家具の配置をシミュレートできるイケアのアプリ「IKEAカタログ」\\
		「IKEAカタログ」というカタログアプリの中に、ARによる家具は一のシミュレーション機能が使われている。この機能を使って、検討中の家具の3D映像を自身の部屋に於いてみて全体の印象を確認することができる。「せっかく買った家具なのに、実際に部屋に置いてみると印象が違った」という、インテリア業界では起こりがちな問題に対する取り組み
%		\cite{DifferenceBetweenVRAndAR}
	\item 教科書AR\\
		東京書籍が開発した「教科書AR」を使って、スマートフォンを教科書にかざすと、そのページに関連した立体がスマートフォンの画面に現れる。教科書だけでは表現できない「動き」も表現することができる。教科書のページそのものをマーカーにして、アプリがそのマーカーを認識したときに立体を描画する仕組みになっている。
%		\cite{TextbookAR}
\end{itemize}
そのほかにも、ポケモンGOなどのゲーム、主に軍事航空分野で応用されている、ディスプレイの向こう側からの光を遮断することなくディスプレイ上に無限遠の点に結像する(パイロットは操縦中遠くの方を見ているため)映像を映しこむヘッドアップディスプレイ、頭部に装着して使用するヘッドマウントディスプレイ(これは非透過の場合VRとなる)、Google Glassなどがある。

\subsection{シェアリングエコノミー}
\subsubsection{シェアリングエコノミーとは}
個人が保有する遊休資産(スキルのような無形のものも含む)の貸し出しを仲介するサービスで、貸主には遊休資産の活用による収入、借主にはそうした遊休資産を所有することなく手軽に利用できるというメリットがある。
%\cite{SoumushoSharingEconomy}
\begin{itemize}
	\item Airbnb 個人・法人を問わず、共有スペース、戸建て住宅、個室から個人が所有する島まで幅広い不動産の貸借が可能\\
		ユーザー間の信頼性を高めるために、過去の利用者による「レビュー評価制度」、写真入り身分証明書などから確認を行う「ID認証」、Facebook等の外部のソーシャルメディアの認証情報を利用する「SNSコネクト」、利用者に起因する損害を補償する「ホスト保証制度」等の機能が導入されている。Airbnb社の研究によると、ホテルのないところに物件があり、一般の旅行者が訪れない地域企業などにお金が落ちる効果が創出されている。
	\item Uber スマートフォンやGPSなどを活用し、移動ニーズのある利用者とドライバーをマッチングさせるサービス\\
		各地域のタクシー会社、ハイヤー会社に加えて、個人のドライバーとも提携をしており、利用者はスマートフォンから廃車の依頼をすることができる。ユーザーが安心活便利に利用できるように「過去の利用者による運転手の評価確認」、「事前に登録したクレジットカードからの運賃の電子決済」、「同乗者との割り勘決済」などの機能を提供している。
\end{itemize}
%\cite{SoumushoSharingEconomy}\\\\
シェアリングエコノミーのサービスは総じてユーザーが安心して利用できるようにする付加価値を多数つけていると感じた。やはりシェアリングエコノミーにおいては個人と個人の信頼関係が特に鍵となるようだ。そこで、オンライン上の様々な活動履歴からユーザーの信頼度を総合的にスコア化する、「Prove Trust」というサービスも存在する。ユーザーの信頼度を、Facebook、地域コミュニティサイトのCraigslist、Airbnb、ビットコイン、結婚恋愛サイトの利用状況に基づき、総合的にスコア化する。評価の内容は、ユーザーが実在するかどうかを評価する「Real Score」、ソーシャルネットワーク上での活動の活発さを評価する「Social Score」、ほかのユーザーからの評価を評価する「Feedback Score」により構成され、これらのスコアを集計して最終的な信頼度がスコア化される。このサービスによって貸主は借主に対する信用を得ることができ、借主はより信用のある貸主を選び、安心してシェアリングエコノミーのサービスを利用することができる。
%\cite{SoumushoSharingEconomy}
\subsection{ブロックチェーン}
\subsubsection{ブロックチェーンとは}
ブロックチェーンはビットコインを構成する中核の技術として、サトシ・ナカモト氏によって考案された分散型コンピュータネットワークであり、ビットコインだけではなく、金融や流通、契約などの分野でも応用できるのではないかと考えられ、注目を集めている。\\
ビットコインによる取引でブロックチェーンは以下のように動作する。
%\cite{BlockChain}
\begin{itemize}
	\item 取引は取引情報を利用者が発信することで始まる。
	\item 利用者は取引情報に暗号署名し、それをネットワークに流す。
	\item 取引情報は同時に分散データベースにも送られる。
	\item 各分散データベースは、受け取った取引情報を自己のデータベースへ書き込む。
	\item データベースへの取引情報の書き込みは、分散データベース間で整合性が取れるよう特定の合意形成メカニズムの下で行われる(マイニング)
	\item 書き込まれた取引情報は過去履歴を含め参照可能となる
\end{itemize}
%\cite{BlockChain}\\\\
これに対し、既存の中央管理型のデータベースの場合は以下のような手順となる。
\begin{itemize}
	\item 利用者は、中央管理者に取引申請を送る。
	\item 中央管理者は申請者と内容を確認し不正があれば排除する。
	\item 中央管理者は、事故のデータベースに取引情報を書き込む。
	\item 中央管理者は取引相手の利用者に取引情報を通知する。
\end{itemize}
%\cite{BlockChain}\\\\
このように、中央管理型のデータベースではデータベースの直接の参照権限は基本的に管理者にしかなく、管理者にデータを改竄されると外部からはわからなくなる。ブロックチェーンでは、同じデータを記録しているコンピュータを世界中に置くことで、データの損失、改竄を防ぐことができる。取引情報をデータベースに追加するときは、世界中のすべてのデータベースを整合性を保って更新させる必要があるので、合意形成(マイニング)と呼ばれる作業を必要とする。そのためデータベースの更新速度が遅いというデメリットがあるが、データベースが一元的に管理されておらず中央管理者がいなくてもデータが安全に保持されるというメリットを持つ。
%\cite{BlockChain}
\subsection{ロボティクス}
\subsubsection{第三の波}
最近はIoTの流れとともに、ロボットを活用したニュースを多く目にするようになった。改めてこれまでの企業や国の施策状況を紐解いていくと、今のロボット業界の流れは、「第三の波」といえるのでは、と考えられる。「第一の波」は産業用ロボットに始まり経済成長を支え、「第二の波」はサービス分野へのロボット適用を推進し、愛・地球博の前後を軸にした機関。今回IoTがあらためて加速している中で、当然高度なデバイスとしてのロボットあるいはロボット技術を活用したソリューションが多く出始めている状況が、今の「第三の波」を形成しているといえる。
%\cite{NTTRobotics}
\subsubsection{今後のロボティクスの存在の在り方}
今後重要となってくるロボットの定義として、掃除用ロボットのように実態を持っており、ユーザーが認識できるビジブル型ロボットと、センサーや空調制御のような実態を見せず、ユーザーに意識させない安コンシャス型ロボットと、仮想的なロボットであり、ユーザーがアバター等を通して認識できるバーチャル型ロボットの3タイプがある。この分類はすでにネットワークロボットの定義として、総務省のプロジェクトを通して何年か前から存在しているものだが、今後出現するロボット技術を活用したサービスの大枠を示す上で改めて注目されている。
%\cite{NTTRobotics}
\subsubsection{Pepper for Bizz}
Pepper for Bizzはソフトバンクが開発した退陣コミュニケーションに特化したロボット「Pepper」の法人向けモデルである。接客や受付などへの企業導入を目的としている。ソフトバンクは、設立30年をむかえた2010年、次の30年はどこへ向かうか、全社的に議論し1つのテーマとして浮かび上がったのがロボット事業への参入だった。個人向けの一般モデルでも、ソフトウェア開発キットを用いてPepperを動作させるアプリケーションを開発できるが、開発にはある程度の基礎知識や経験が必要で、プログラミングの基礎知識が不要で誰もがより簡単にPepperの仕事を作ることができるよう、Pepper for Bizを開発した。
%\cite{Pepper}
\subsection{ビッグデータ（データ分析）}
\subsubsection{ビッグデータとは}
ビッグデータとは「事業の拡大に役立つ膨大なデータ」のことを指す。ビッグデータを活用している企業は膨大なデータを管理、分析して売り上げを上げたり、仕入れの最適化などを行っている。ビッグデータの活用によって得られる価値として、以下のようなものが挙げられる。
\begin{itemize}
	\item 業務フローの最適化によるコストの削減
	\item 商品の組み合わせによる売り上げの最大化
	\item 未来の売り上げ予測
	\item 事業の問題や課題個所の特定
	\item 新たな仮説や可能性の発見
\end{itemize}
%\cite{BigData}
\subsubsection{なぜ今ビッグデータが話題になっているのか}
テクノロジーの進化によって以下のようなことが可能となり、膨大なデータを使ってできるとこが増えたため、ビッグデータが今話題となっている。
\begin{itemize}
	\item 今まで取得できなかったデータを取得できるようになった。
	\item 今まで記録や保存が難しかった量のデータを管理できるようになった。
	\item 今まで活用できなかった量のデータを処理できるようになった。
\end{itemize}
%\cite{BigData}
\subsubsection{従来のデータ分析との違い}
従来のデータ分析では、そもそも膨大なデータを集めること自体相当なコストがかかる事であったため、これまでの知見を活かし、仮説を立て、その仮説を示す仕組みを仮想的に構築するモデリングを行い、そのモデリングの中で重要だと思われる場所にセンサーデバイスを設置し、それをサンプルデータとして測定し、そのデータから現実に起きた事象の母集団を推定する必要があった。しかし、より膨大な量のデータを処理することができるようになった今では、仮説、モデリング、サンプリングをする必要はなく、母集団をそのままデータとして得ることができるようになった。
%\cite{BigDataSample1}
\subsubsection{事例}
\paragraph{事例1}
清涼飲料メーカーのダイドードリンコはZの法則にしたがい主力シリーズの「ブレンドシリーズ」を自動販売機の左上に配置していたが、全ての自動販売機に人の目線を認識するアイトラッキングセンサーを取り付け調査したところ、人の視線は自動販売機の下段に集中していることが分かり、下段にブレンドシリーズを配置したところ売り上げが前年比1.2\%増となった。
%\cite{BigData}
このように、ビッグデータの活用の多くは顧客のニーズ、需要、行動をより細かく知るという方法をとっている。
\paragraph{事例2}
全米屈指の犯罪都市であるカリフォルニア州オークランドでは、犯罪の種類別、日にち別、時間別にフィルタが可能で色別に確認することができ、その日その時に危険な場所を特定できるため、観光客や市民などの利用者が危険を避けることが可能となる。
%\cite{BigData}
これは逆に犯罪者にも利用されかねないと思った。
\paragraph{事例3}
日立製作所中央研究所主管研究長の矢野和男氏は、受注率の異なる二つのコールセンターを比較するために、両コールセンターの人員に名札が他のセンサーを装着してもらい、体の動きを計測したところ、受注率が高い方のコールセンターの人員の方が休憩中の体の動きが活発で雑談がよく弾んでいることが判明し、スタッフ同士の休憩中のコミュニケーションを活発化させるような施策を一年間適用したところ、コールセンターの売り上げが27\%増加した。
%\cite{BigDataSample3}
このように、顧客のニーズだけでなく、企業の従業員の行動に対してビッグデータを活用することによって売り上げを伸ばし、従業員もより快適に労働に従事することができるようになったというケースもある。
\end{comment}
\section{新しいビジネスの可能性}
ARやVR、特にGoogle Glassのようなデバイスを用いてできることをいくつか考えてみた。
\begin{itemize}
	\item 試験中に問題文を読み取り答えを表示するようなアプリ
	\item ARに視覚だけではなく聴覚や嗅覚を再現できるようにする。COOKPADなどで活用できるかもしれない。
	\item 人の顔を認識してネットからその人の情報を取ってきて表示する。
	\item 外国人の音声を認識して、それが何語なのかを表示する。またユーザーが指定した言語に翻訳して表示する。
	\item ARをより使いやすくするために、左右の眼球の角度から今その人の視線はどこにあるのか、どの距離のものを見ているかを判断し、それに合わせて画像を表示し、目の負担を減らす。
\end{itemize}
